## APPENDIX: Methodology Details

**Step 1: Source & Citation Discovery**
- **29 trust nodes evaluated** across 6 categories (Knowledge Graphs, Review Platforms, Directories, Company Profiles, News/PR, Seed Sites)
- **Web search used** to verify presence (Google, legal tech directories, review platforms)
- **Coverage percentage calculated:** Present nodes / Total possible nodes (3/29 = 10.3%)
- **Alternative trust nodes tracked:** Additional 8 sources beyond core 29 (ZoomInfo, D&B, BBB, Datanyze, Glassdoor, Facebook, ALA, Founder Book)

**Step 2: Citation Quality Scoring**
- **5 dimensions scored** (0-10 scale):
  1. Authority - Domain authority (DA/DR), editorial standards, platform reputation
  2. Data Structure - Schema.org markup, structured data, machine-readable format
  3. Brand Alignment - Accurate representation, sentiment, name consistency
  4. Freshness - Publication date, last updated, content recency
  5. Cross-Link Signals - References to/from other trust nodes, citation network strength
- **Citations fetched and analyzed:** 11 present trust nodes
- **Composite scores calculated:** Average across all 5 dimensions per citation, then overall average (7.9/10)
- **Quality distribution:** High (8-10), Medium (7-8), Low (0-7) buckets

**Step 3: LLM Response Evaluation**
- **3 platforms tested:** Perplexity (API), ChatGPT (browser automation), Gemini (browser automation)
- **Query taxonomy applied:**
  - **Perplexity:** 3 query types (Evaluative, Comparative, Brand-Specific)
  - **ChatGPT:** 1 query (Evaluative - browser automation limitations)
  - **Gemini:** 1 query (Evaluative - browser automation limitations)
- **Position tracking:** Numerical rank if listed (#1-10), "Not mentioned" if absent
- **Citation mapping:** URLs cited by each platform extracted, cross-referenced with Step 2 citations
- **Browser automation:** @playwright-cleanup invoked before each browser agent, 20-second wait times, proper browser closure

**Step 4: Dashboard Synthesis**
- **Cross-step analysis:** Connected trust node gaps (Step 1) ‚Üí citation quality issues (Step 2) ‚Üí LLM visibility problems (Step 3)
- **Strategic prioritization:** Ranked actions by impact (trust node establishment > citation quality improvement > content creation)
- **Competitive benchmarking:** Analyzed why competitors (TimeSolv, Accurate Legal Billing, The Legal Billing Group) outrank B2
- **Causal chain mapping:** Documented specific examples of trust node ‚Üí citation ‚Üí LLM visibility connections
