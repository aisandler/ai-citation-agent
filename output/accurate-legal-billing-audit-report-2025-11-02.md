## APPENDIX: Methodology Details

**Step 1: Source & Citation Discovery**
- **29 trust nodes evaluated** across 6 categories (Knowledge Graphs, Review Platforms, Directories, Company Profiles, News/PR, Seed Sites)
- **Web search verification:** Google Search, Bing, DuckDuckGo used to verify presence
- **Coverage percentage calculated:** 11 present / 29 total = 38% coverage
- **Quality scoring:** Each present node scored 0-10 based on completeness, accuracy, freshness
- **Critical gaps identified:** Missing nodes prioritized by LLM impact (e.g., Wikipedia > Trustpilot)

**Step 2: Citation Quality Scoring**
- **9 citations analyzed** from Step 1 trust nodes (LinkedIn, Crunchbase, G2, Capterra, Software Advice, Pitchbook, Inc.com, Product Hunt, official website)
- **5 dimensions scored (0-10 scale):**
  1. Authority - Domain authority (DA 50+), editorial standards, industry recognition
  2. Data Structure - Schema.org markup (JSON-LD, Microdata), structured data quality
  3. Brand Alignment - Accurate representation, sentiment analysis, positioning consistency
  4. Freshness - Publication date, last updated timestamp, content recency
  5. Cross-Link Signals - References to/from other trust nodes, citation network strength
- **Composite scores calculated:** Average of 5 dimensions for overall quality (7.6/10 average)
- **Distribution analysis:** High (8-10), Medium (5-7), Low (0-4) quality tiers
- **Citation fetching:** Live URL verification, schema markup inspection, cross-reference checking

**Step 3: LLM Response Evaluation**
- **3 platforms tested:** Perplexity (API-based), ChatGPT (browser automation), Gemini (browser automation)
- **Query taxonomy applied:**
  - Evaluative: "What are the top [category] in 2025?" - All 3 platforms
  - Comparative: "Best [category] for [use case]" - Perplexity only (time optimization)
  - Brand-Specific: "[Brand] reviews and credentials" - Perplexity only
- **Position tracking:** Ranked position (#1-10), "mentioned but not ranked," or "not mentioned"
- **Citation mapping:** Extracted source URLs, matched against Step 2 citations, identified new sources
- **Competitor analysis:** Top 3-8 competitors per platform, positions tracked, citation sources mapped
- **Browser automation safety:**
  - Playwright cleanup invoked before ChatGPT and Gemini agents (prevents zombie browser processes)
  - 20-second wait times for web search completion (optimized from 30s)
  - Proper browser closure after extraction
  - 90-second timeout failsafe

**Step 4: Dashboard Synthesis**
- **Cross-step analysis:** Connected findings from Steps 1, 2, 3 to identify causal chains
- **Strategic prioritization:** Immediate (30 days), Strategic (90 days), Long-term (6-12 months) based on impact vs. effort
- **Competitive benchmarking:** Compared brand's trust node coverage, citation quality, LLM rankings against top competitors
- **Citation influence mapping:** Identified which Step 2 citations (or missing citations) drove Step 3 LLM visibility gaps
- **ROI estimation:** Calculated potential visibility improvement from each priority action
