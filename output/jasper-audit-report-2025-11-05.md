- Composite scores calculated: Average across all dimensions for each citation, then average across all citations for overall quality
- Distribution tracking: High (8-10), Medium (5-7), Low (0-4) quality tiers

- 3 platforms tested: Perplexity (API-based), ChatGPT (browser automation - incomplete), Gemini (browser automation)

  - ChatGPT: 1 query (Evaluative) via browser - authentication blocked evaluation
  - Gemini: 1 query (Evaluative) via browser - successfully completed
- Position tracking: Numerical ranks where applicable, categorical positioning where not
- Citation mapping: All source URLs cited by each platform extracted and matched to Step 2 citation URLs
- Competitive analysis: Top competitors identified and ranked for each query

**Step 4: Dashboard Synthesis**
- Cross-step analysis connecting trust node gaps ‚Üí citation quality weaknesses ‚Üí LLM visibility patterns
- Strategic prioritization based on impact (knowledge graph highest ROI) and effort (quick wins like URL fixes)
- Competitive benchmarking vs. ChatGPT, Claude, HubSpot AI across Perplexity and Gemini
